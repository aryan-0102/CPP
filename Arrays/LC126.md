## 🤔 My Approach & Intuition
XOR outputs zero for the same values thus the only value remaining would be the answer.

## ⚙️ Complexity Analysis

* **Time Complexity:** $O(N)$
    * **Space Complexity:** $O(1)$
    ---

## 💻 Code Implementation

``````cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int sol =0;
        for(auto a:nums){
            sol ^= a ;
        };
        return sol ;
    }
};
